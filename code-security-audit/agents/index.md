# Agent 索引

## top-down

- **适用场景**: 用户指定了入口点（接口/API handler/CLI 命令/导出函数），需要从入口向下追踪调用链，发现安全漏洞。
- **能力简介**: 入口函数识别、调用链追踪、参数传播分析、攻击面映射、漏洞定位。
- **预调查输入**: 主 skill 会提供入口的精确文件路径、行号、函数签名、路由注册信息。subagent 可直接使用，无需重复定位。
- **文件**: `agents/top-down.md`

## bottom-up

- **适用场景**: 用户指定了危险操作点（sink/SAST 告警/可疑调用），需要从该点向上反查数据流，确认是否真实可利用。
- **能力简介**: Sink 点分类、反向数据流追踪、输入源识别、利用条件判断、误报排除。
- **预调查输入**: 主 skill 会提供 sink 点的精确文件路径、行号、函数名及周边代码上下文。subagent 可直接使用，无需重复定位。
- **文件**: `agents/bottom-up.md`
